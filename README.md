## üöÄ Quick Start Guide

### **Step 1: Environment Setup**
```bash
# Clone the repository
git clone https://github.com/yourusername/c-programming-handbook.git
cd c-programming-handbook

# Create projects directory
mkdir projects
cd projects
```

### **Step 2: First C Program**
```c
// hello.c
#include <stdio.h>

int main() {
    printf("Hello, C Programming! \\n");
    printf("Namaste, C Programming! \\n");
    return 0;
}
```

### **Step 3: Compile & Run**
```bash
# Compile
gcc hello.c -o hello

# Run
./hello
```
